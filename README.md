# ① 課題名

ミロク 🧞‍♀️ のブックアーカイブ

## ② 課題内容（どんな作品か）

- 「気になるこの本！でも読む暇がない！」って大丈夫 👌<br>
  「ミロクのブックアーカイブ」に読みたい気持ちだけ保存 ✨<br>
  後から感想も追記できちゃう 💫 アタイ天才でしょ 🧞‍♀️

- あなたは人生の中で、幾多もの読みたい本に出逢います。しかし、全ての本を読むことはできますか？積読できますか？泣く泣く忘れてしまうこともあるのではないでしょうか？<br>そんな時はとりあえず、読みたい本の「タイトル」と「読みたい気持ち」だけ保存しておけば、安心できるのではないでしょうか。<br>JS の localstorage と Google Books APIs を活用し、この 2 点だけ保存できる作品を作りました。ついでに、感想を随時保存できるようにしています。

## ⑤ 工夫した点・こだわった点

- あくまでローカル環境で完結する情報伝達ですが、星のアニメーションをパケット送信に見立てて、あたかもサーバー間で情報が伝達しているような雰囲気をつくってみました。
- localstorage の value にタイムスタンプを付加する仕組みで、図書カードのように時系列を残せるようにしました。

## ⑥ 難しかった点・次回トライしたいこと（又は機能）

- タイトル欄、興味欄が空白でも、登録ボタンを押すと星のアニメーションが発生してしまう不具合があります。
- 登録順にサムネイルを発生させることができません（Google Books APIs の設定で工夫が必要？）。
- 登録した本の数に応じてゲージが伸びていく、インジケータバーを実装してみたかったです。
- レスポンシブデザインに着手する余裕はちょっとありませんでした。
- 可読性に配慮する余裕がなく、不要な記述や無駄な書き方であふれていそうな現状です。

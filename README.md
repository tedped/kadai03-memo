## ⑤ 工夫した点・こだわった点

- localstorage を活用した、あくまでローカル環境で完結する情報伝達だが、星のアニメーションをパケット送信に見立てて、あたかもサーバー間で情報が伝達しているような様子を演出した。
- 感想を何回でも追加できるようにするとともに、感想を残すたび localstorage の「value」にタイムスタンプを付加。図書カードのように履歴を残すことができるようにした。
- Google Books APIs を使用。認証キー不要・無料。導入障壁が低いおかげで、このアイデアを思いつき、制作に挑戦できた。感謝。

## ⑥ 難しかった点・次回トライしたいこと（又は機能）

- 可読性に配慮する余裕がない、おそらく不要な記述や無駄な書き方であふれていると思う
- タイトル欄、興味欄が空白でも、登録ボタンを押すと星のアニメーションが発生してしまう不具合を解決できなかった
- 登録順にサムネイルを発生させることができなかった
- 最初に情報を登録する時のタイムスタンプ機能を実装できなかった（localstorage の「key」にタイムスタンプを付加できなかった）
- 登録した本の数に応じてゲージが伸びていく、インジケータバーを実装してみたかった
- UI をとにかくかわいくしたかったがちょっと余裕はなかった
- レスポンシブデザインに着手する余裕はちょっとなかった
